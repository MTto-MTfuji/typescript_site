TypeScript道場

「TypeScript道場」は、JavaScriptとTypeScriptを体系的に学べるオンライン学習サイトです。  
ブラウザがあれば、基礎文法から実務で使うフレームワークまで、段階的に学習できます。

- 対象者: プログラミング初心者 / JavaScriptを触ったことはあるが型の概念に慣れたい方 / TypeScriptを実務レベルで使えるようになりたい方
- 学習スタイル: 読むだけでなく、コード例・練習問題・進捗管理・メモ機能を使って「手を動かしながら」学ぶスタイル
- 利用料金: 無料（個人学習用サイト）

特徴

-  完全網羅: JavaScriptとTypeScriptの全機能を体系的に学習
-  初心者歓迎: 基礎から丁寧に解説し、段階的にレベルアップ
-  実践的: React、Vue、Next.jsなど実際のフレームワークでの使い方も学べる
-  インタラクティブ: コード例をコピーしてすぐに試せる
-  UI: モダンで使いやすいインターフェース

学習内容

JavaScript
- 基礎編: 変数、関数、配列、オブジェクトなどの基本構文
- 中級編: ES6+の機能、非同期処理、クラス、モジュール
- 上級編: デザインパターン、メタプログラミング、パフォーマンス最適化

TypeScript
- 基礎編: 型システム、インターフェース、クラス、型ガード
- 中級編: ジェネリクス、ユーティリティ型、条件型、マップ型
- 上級編: 高度な型操作、型レベルプログラミング、ブランド型

学習の進め方

1. JavaScript基礎編から始める（プログラミング初心者の場合）
2. JavaScript中級編でES6+の機能を学ぶ
3. TypeScript基礎編で型システムを理解する
4. TypeScript中級編でより高度な型操作を学ぶ
5. フレームワーク編で実践的な開発を学ぶ

各セクションは独立しているため、ご自身のレベルに合わせて自由に学習できます。
